Liquid templating language

Created by Shopify

  • Also used by:
  • Harmony
  • LocomotiveCMS
  • Siteleaf

Liquid example:

{% if collection.products.size > 0 %}
  {% for product in collection.products %}
    <h1>{{ product.title }}</h1>
  {% endfor %}
{% endif %}

If statements:

{% if moose %}
  ...
{% elsif monkey %}
  ...
{% endif %}

Loops:

{% for moose in monkey %}
  ...
{% endfor %}

Variable assignment:

{% assign moose = monkey %}

Liquid output:

{{ page_title }}

Liquid objects:

{{ product.title }}
{{ product.description }}
{{ product.images }} # is an array

Liquid filters:

{{ 5 | times:5 }}
=> 25

{{ ‘my moose’ | replace: ‘moose’, ‘monkey’ }}
=> my monkey

{{ ‘moose/monkey’ | split: ‘/’ }}
=> [‘moose’,’monkey’]

Filter chaining:

{{ product.images | first | img_tag | link_to: product.url }}

<a href="/product/my-product">
  <img src="/images/my-product.jpg" alt="My product">
</a>
